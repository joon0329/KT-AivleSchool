# KT AIVLE School 학습 기록

> 본 저장소는 **KT AIVLE School** 과정에서 학습한 내용을 카테고리별로 정리하고,
> 복습 및 포트폴리오 활용을 목적으로 운영됩니다.

KT AIVLE School 과정에 참여하며 **AI 분석/생성 기술**, **백엔드·프론트엔드 서비스 개발**,
그리고 **클라우드 환경에 대한 기초 이해**까지 단계적으로 학습한 내용을 정리한 저장소입니다.

* **주최/운영**: KT AIVLE School
* **교육 방식**: 실습 중심의 풀타임 교육 과정
* **과정 개요**:
  분석형 AI부터 생성형 AI, 백엔드·프론트엔드 개발, 클라우드 환경 이해까지

##  주요 기술 스택

* **AI / Data**: Python, Pandas, Machine Learning, Deep Learning, Generative AI(RAG, Agent)
* **Computer Vision**: CNN, Object Detection(YOLO), Multimodal AI
* **Backend**: Java, Spring Framework, JPA, REST API, Spring Security
* **Frontend**: HTML, CSS, JavaScript, React, Next.js
* **Cloud**: Cloud Computing 개념, 네트워크 및 인프라 기초

---

##  커리큘럼 상세

### 1. 데이터 분석 및 머신러닝

* 데이터 분석 프로세스 및 탐색적 데이터 분석(EDA)
* 머신러닝 모델링 및 성능 평가
* 회귀, 분류, 앙상블 알고리즘 학습

### 2. 딥러닝 및 컴퓨터 비전

* 딥러닝 기본 구조 이해
* CNN 기반 분류 모델링
* Object Detection(YOLO) 및 시각 지능 활용

### 3. 생성형 AI 및 AI Agent

* 자연어 처리 및 프롬프트 엔지니어링
* RAG(Retrieval-Augmented Generation) 구조 이해
* VectorDB 활용
* LangChain / LangGraph 기반 AI Agent 설계
* 멀티모달 AI 적용

### 4. 백엔드 개발

* Java 기초 문법 및 객체지향 프로그래밍
* Spring Framework 기반 웹 애플리케이션 개발
* MVC 아키텍처 및 RESTful API 설계
* JPA를 활용한 데이터 관리
* 인증·인가(Spring Security)

### 5. 프론트엔드 개발

* HTML / CSS / JavaScript 기초
* React 컴포넌트 기반 개발
* 상태 관리 및 라우팅
* Next.js 기반 서버 통신
* JWT 인증 기반 로그인 구현

### 6. 클라우드 및 서비스 운영 기초

* 클라우드 등장 배경과 현대 IT 환경 이해
* 클라우드 네트워크 구성 개념
* 컴퓨팅 서비스 구조 이해
* 서비스 운영 관점의 인프라 기초

### 7. 실전 프로젝트

* 데이터 분석 및 예측 프로젝트
* 생성형 AI Agent 프로젝트
* 멀티모달 AI 응용 프로젝트
* 풀스택 웹 서비스 개발 프로젝트
